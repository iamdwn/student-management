# Student Management

A system to manage student, subject and score of the student.

## Technologies Used

Spring Boot: Backend framework for building the game application.
Spring Security: For handling user authentication and authorization.

## Features

CRUD of Student.
CRUD of Subject.
CRUD of Score.
